A propos de moi

Bonjour et bienvenue sur mon site !

Je m'appelle Adèle et je suis une jeune développeuse web passionnée. Depuis toujours, je suis curieuse et intéressée par l'informatique. C'est pourquoi j'ai décidé de consolider mes connaissances en suivant la formation de développeur intégrateur web proposée par OpenClassrooms. Aujourd'hui, je suis à la recherche de mon premier emploi dans ce domaine. Grâce à ma formation, j'ai acquis les compétences nécessaires pour développer des sites web modernes et performants et j'ai confirmé mon désir de travailler dans le développement web.

Mes projets

Ohmyfood !

Création d'un site web en utilisant les langages HTML et CSS, en se basant sur une maquette. Le préprocesseur CSS Sass est utilisé pour la mise en forme et la gestion du style. Le site est également animé en utilisant des animations CSS, afin d'ajouter de l'interactivité et de la dynamique à l'expérience utilisateur.

Booki

Création d'un site web à partir d'une maquette en utilisant les langages HTML et CSS. Le site a été conçu pour être adaptatif aux différents appareils, grâce à un design responsive.

Kanap

Utilisation de Javascript pour un site d'e-commerce. Les éléments nécessaires à l'affichage du site ont été récupérés en utilisant des appels à une API, puis affichés sur la page web. Pour stocker les données localement, la fonctionnalité localStorage à été utilisée.

Kasa

Intégration d'une maquette de site web en utilisant les langages HTML et CSS et utilisation des composants de React. Le site est également conçu pour être responsive. Pour faciliter la mise en forme et la gestion du style, le préprocesseur CSS Sass est utilisé.
